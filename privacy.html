Privacy Policy - KeepWeb

KeepWeb is a browser extension whose sole purpose is to help you save and organize links and notes from web pages. This policy explains how user data is handled.

Data Collected

- URLs and titles of web pages that the user saves.
- Text or selected snippets to create notes.

How Data is Used

Exclusively to allow the user to save and retrieve links and notes within the extension.

Data Sharing

Data is not shared with third parties and remains local on the user's device or, if Chrome sync is used, within the secure Chrome Storage Sync ecosystem.

Security

Data is stored securely and is only accessible by the user.
